Most common programming languages used in India ?¶

A data analysis using Stackoverflow’s 2018 and 2019 Annual Developer Survey.

Introduction¶

With the advent of new areas in India such as Artificial Intelligence and Data Science, many programming languages that were poorly spoken and used are now experiencing a rise in popularity.

You may have already heard “Python is the programming language of the moment” or “Functional Paradigm is much better than Object Oriented”.

In this article, we are going to analyze real data to verify if these programming languages are really being used India or if they are just rumors.

For this, we are going to use data from Stackoverflow’s 2018 and 2019 Annual Developer Survey.

Every year, Stack Overflow conducts a massive survey of people on the site, covering all sorts of information like programming languages, jobs, code style and various other information.

Every year, Stack Overflow conducts a massive survey of people on it’s website, covering all sorts of information like programming languages, jobs, code style and more.
